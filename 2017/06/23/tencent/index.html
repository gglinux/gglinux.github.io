公司那些事

| |

毕业一年了，上上下下经历了不少公司。自己创过业，也在B轮公司待过，也在D轮做过事，另外也在上市公司做过螺丝钉。林林总总算下来，也有五家公司了。也是挺折腾的。经常有人会觉得，这小伙沉不下来。其实对应聘者而言，反反复复更换工作，当然有其不好之处。

但从另一方面考虑有些人为什么会频繁更换工作呢？人往往喜欢待在舒适区，待久了一个地方就懒的去更换巢穴。频繁的跳出舒适区，也是对自己发展现状不满意的宣泄。这也是对笔者频繁更换工作的一种解释吧。我是觉得适当时机跳出舒适区，是对个人职业发展的追求。

入职腾讯也有两周了，这期间经常有人问我腾讯的面试情况。在不违反公司制度的前提下，我会简单介绍有关问题。其次会比较下我目前待过几家公司的特点。

腾讯面试

经历过三次腾讯面试（手动捂脸），第一次是2015年校招面试IEG后台开发岗，结果是到HR面终止。校招面试流程为：网申-》笔试-》一轮专业面(有笔试)-》二轮专业面-》HR面。面经我就不细述了，牛客网有很多。自己之前也写过一篇。

第二次是2016年大学毕业，从美图离职那次。面试的SNG腾讯云的运营开发岗，结果是一面结束。面试流程为：拉勾-》面试(有笔试)-》结束。社招，有两年工作经验的面试者，应聘的成功率会高很多。不建议刚毕业通过社招途径进入。

第三次是2017年工作一年后。面试的TEG后台开发岗，顺利入职。面试流程为：内推-》一面专业面（有笔试）-》二面专业面-》3面HR面-》沟通薪资-》背景调查-》发放offer-》确定入职时间-》入职。专业一面会问的比较详细，如果一面专业面搞定，后面基本过过流程，所以主要是一面。一面侧重点，一是项目相关，另外是专业基础。社招，项目相关的会比较多，注意所做项目的深度。这些都是笔者的一些感受，每个岗位可能都不一样，不可以偏概全。

腾讯生活

说几个感受比较深刻的吧。腾讯有内部食堂，价格实惠，笔者所在食堂味道一般（和工作地点有关）。深圳地区早晚有班车，通勤方便，笔者上下班基本都坐班车了。超过8点，有18元晚餐券。开发设备配置较好，两台电脑。想内推的可发简历到：gglinux@163.com。

特点比较

以下评价为个人观点，每家公司都会变化，不代表100%实际情况。A-》最好、B-》其次、C-》较差、D-》比较差，四个等级。建议职场新人能去大公司，还是去大公司。
